## 加密不可用
<Image
  src={encryption}
  alt="A description of my image."
  width="300"
  height="150"
/>

使用以下命令在客户端运行时找到内核所在位置
```shell
sudo ps -ef | grep mihomo
```

执行以下命令手动给内核授权即可解决，请注意更改内核路径

```shell
sudo setcap cap_net_bind_service,cap_net_admin,cap_sys_ptrace,cap_dac_read_search,cap_dac_override,cap_net_raw=+ep /path/to/mihomo-core
```
