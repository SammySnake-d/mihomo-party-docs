通过 [Github Release](https://github.com/pompurin404/mihomo-party/releases/latest) 下载 rpm 包，使用 rpm/dnf/yum 安装

<Tabs>
<TabItem label="rpm">

```shell
sudo rpm -ih /path/to/mihomo-party-linux-xxx.rpm
```

</TabItem>
<TabItem label="dnf">

```shell
sudo dnf install /path/to/mihomo-party-linux-xxx.rpm
```

</TabItem>
<TabItem label="yum">

```shell
sudo yum localinstall /path/to/mihomo-party-linux-xxx.rpm
```

</TabItem>
</Tabs>
