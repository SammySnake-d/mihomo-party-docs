通过 [Github Release](https://github.com/mihomo-party-org/mihomo-party/releases/latest) 下载 deb 包，使用 apt/dpkg 安装

<Tabs>
<TabItem label="apt">

```shell
sudo apt install -y /path/to/mihomo-party-linux-xxx.deb
```

</TabItem>
<TabItem label="dpkg">

```shell
sudo dpkg -i /path/to/mihomo-party-linux-xxx.deb
```

</TabItem>
</Tabs>
